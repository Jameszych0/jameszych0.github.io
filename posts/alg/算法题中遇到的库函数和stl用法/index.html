算法题中遇到的库函数和stl用法

August 16, 2025

isalnum()

125. 验证回文串

如果在将所有大写字符转换为小写字符、并移除所有非字母数字字符之后，短语正着读和反着读都一样。则可以认为该短语是一个 回文串 。

字母和数字都属于字母数字字符。

在本题中我们需要判断字符串中字符是否是字母数字字符，可以使用isalnum()进行判断

#include <ctype.h>

int main() {
    ...
   	if (isalnum(str[i]))
}

ctype.h中还有一些很好用的函数

  • isalpha()
  • isdigit()
  • tolower()
  • toupp
