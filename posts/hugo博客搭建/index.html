多设备上传 #

如果后期需要使用其他电脑进行文章的上传，可以按照本节的描述操作。

所以，我们只需要将该项目克隆下来即可实现多设备上传。

git clone https://github.com/Jameszych0/hugo_blog.git

注：若多设备使用，提交前需使用git pull以保证项目一致
